# John J Carey

Home theater automation, Home Assistant custom integrations, and Unfolded Circle Remote integrations.


## 🏠 Home Assistant Integrations

Custom Home Assistant integrations for home theater and AV devices.

### [Lumagen Home Assistant Integration](https://github.com/johncarey70/lumagen-ha)

Home Assistant custom integration for Lumagen Radiance Pro video processors.

- UI-based setup
- TCP serial and USB serial support
- Media player, remote, buttons, switches, and sensors
- Input/source control and Lumagen status reporting

### [HDFury Home Assistant Integration](https://github.com/johncarey70/hdfury-ha)

Home Assistant custom integration for HDFury devices.

- Device status monitoring
- Input/output selection support
- REST-based communication
- Useful for HDMI switching and signal management setups

### [Seymour Home Assistant Integration](https://github.com/johncarey70/seymour-ha)

Home Assistant custom integration for Seymour motorized masking systems.

- Motorized screen/masking control
- Position and ratio control
- Home Assistant service support
- Designed for home theater automation workflows

## 🎛️ Unfolded Circle Remote Integrations

Custom integrations for the Unfolded Circle Remote platform.

### [Lumagen Unfolded Circle Integration](https://github.com/johncarey70/uc-integration-lumagen)

Unfolded Circle integration for Lumagen Radiance Pro video processors.

- Remote control commands
- Media player support
- Source/input control
- Home theater activity integration

### [Kaleidescape Unfolded Circle Integration](https://github.com/johncarey70/uc-integration-kaleidescape)

Unfolded Circle integration for Kaleidescape systems.

- Player and server control
- Browse, search, and play support
- Playback status and metadata updates
- Cover art and title handling

### [Trinnov Unfolded Circle Integration](https://github.com/johncarey70/uc-integration-trinnov)

Unfolded Circle integration for Trinnov Altitude processors.

- Preset and source/profile control
- Volume and mute control
- Status feedback
- Designed for advanced home theater control systems
